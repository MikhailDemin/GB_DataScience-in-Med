Курсовой проект,
БД сайта "Кинориум" https://rus.kinorium.com/

Сайт представляет собой сервис пользовательских списков и рейтингов к различным фильмам. Создав профиль можно создавать списки из актеров/режиссеров/фильмов и давать им свои оценки. Можно добавлять анонсированные фильмы ("Скоро", "Скоро Онлайн", "Премьера", "В кино"). Сервис напомнит орелизе картины корую вы ждете. Также предупредит о релизе на онлайн-площадках и укажет где можно посмотреть. Можно написать свои комментарии, есть лента друзей и лента новостей кино, можно подписываться на друзей, видеть что они смотрят, обмениваться мнениями. Также есть возможность шерить в соцсети.
В разделе "сериалы" можно давать оценки отдельным сериям и сезонам. Также на странице актера/режиссерав/фильма подтягивается рейтинг ImDB, Rotten Tomatoes и рейтинг выставленный по оценкам пользователей сайта. 
В профиле автоматически формируются списки "Смотрю" (для сериалов), "Просмотрено", "Буду смотреть", "Не буду смотреть", в зависимости от того что вы выбрали на странице фильма. Также можно формировать свои списки, искать похожие фильмы. Есть фильтрация поиска.
 
Есть мобильное приложение для Android и iOS. У сервиса есть API и расширение для браузера.
